# SymfonyDocker app

## Tech stack
- Docker 
- PHP 7.4
- MySQL 
- Nginx
- Symfony 5

## Info

It is composed by 3 containers, separatly.

- `nginx` - webserver
- `php` - PHP-FPM 7.4
- `db` - MySQL 8.0.

## Installation

1. Clone this repository via cmd `git clone <ssh url>`

2. Run `docker-compose up -d`
